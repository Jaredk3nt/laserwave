### LaserWave High Contrast (WCAG AAA)

![screenshot](screenshot-high-contrast.png)

**Currently supports:**

- Python
- Javascript
  - React
- Typescript
  - Angular
- Ruby
- Markdown
- Java
- C#
- Dart
- F#

If you find any missing highlighting or want a new language supported open an issue with an example file from the language üòÄ
